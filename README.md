## Install
### Plug

```
Plug 'kamykn/dark-theme.vim'
```

### NeoBundle

```
NeoBundle 'kamykn/dark-theme.vim'
```

## .vimrc
Please add to your .vimrc:
```
colorscheme darktheme
```
